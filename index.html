Technical Implementation

The component utilizes an isolated Shadow DOM to ensure zero CSS collision. Below is a standard implementation example for an authorized reveal workflow.

implementation.js 
// Reference the custom element
const spoiler = document.querySelector('particle-spoiler');

// Block user interaction until authorized
spoiler.locked = true;

// Triggered via custom app logic (e.g., Auth Success)
function unlockContent() {
  spoiler.locked = false;
  spoiler.reveal();
}

// Event handling
spoiler.addEventListener('reveal', () => {
  const state = spoiler.isRevealed;
  console.log(`Content Exposure: ${state}`);
});
